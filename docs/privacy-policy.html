Política de Privacidade

Esta Política de Privacidade descreve como tratamos os dados dos usuários que interagem com nosso assistente virtual (bot) por meio do WhatsApp.

1. Coleta de Dados

Nosso sistema pode coletar as seguintes informações dos usuários:

  • Nome e sobrenome (quando informados);
  • Número de telefone;
  • Histórico de mensagens trocadas com o bot;
  • Dados de navegação e comportamento dentro da conversa, com objetivo de melhoria da experiência.

2. Uso dos Dados

Os dados coletados são utilizados para:

  • Responder às mensagens dos usuários de forma automática;
  • Realizar atendimentos, agendamentos, vendas e suporte;
  • Personalizar a conversa e recomendar produtos/serviços;
  • Aprimorar o desempenho do bot, com base nos dados agregados de conversas.

3. Compartilhamento de Dados

Não compartilhamos dados pessoais dos usuários com terceiros, exceto:

  • Quando exigido por lei ou decisão judicial;
  • Para cumprimento de obrigações legais com parceiros (como o WhatsApp/Facebook);
  • Para provedores de tecnologia essenciais à operação do sistema (ex: servidores, plataformas de IA).

4. Segurança

Adotamos medidas de segurança para proteger as informações coletadas contra acessos não autorizados, alteração ou destruição indevida.

5. Armazenamento

Os dados são armazenados de forma segura, em servidores protegidos, pelo prazo necessário para cumprimento das finalidades descritas nesta política.

6. Direitos do Usuário

Nos termos da Lei Geral de Proteção de Dados (LGPD), o usuário pode solicitar:

  • Acesso aos seus dados;
  • Correção de dados incorretos;
  • Exclusão de seus dados do nosso sistema, quando aplicável.
